# AUR-helper
A simple app that makes installing packages by AUR faster
## How to use it
Download the python file and type the command
```bash
python3 main.py -i PACKAGE_YOU_WANT
```
You can search packages using
```bash
python3 main.py -f PACKAGE_YOU_WANT
```
